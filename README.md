# 📈 Stock Market Analytics

## Overview
A Python project that analyzes historical stock market data using Yahoo Finance.

## Features
- Historical Price Analysis
- Closing Price Visualization
- 50-Day Moving Average
- Trading Volume Analysis
- Candlestick Chart
- RSI Indicator
- MACD Indicator

## Technologies
- Python
- Pandas
- Plotly
- yFinance
- Google Colab

## Sample Stock
Apple (AAPL)
